# scala-cli-millbuild-lazylogging-hello-world

## Description
A POC for mill and scala.
Uses logback and lazy logging
through ivyDeps.

## Tech stack
- scala
- mill

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
